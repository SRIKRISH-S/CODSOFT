## 🚀 Quick Start

### Task 1 — Chatbot (Rule-Based)
```bash
cd task1_AI_chatbot
pip install -r requirements.txt
streamlit run app.py
```

### Task 2 — Tic-Tac-Toe AI (Minimax + Alpha-Beta)
```bash
cd task2_tic_tac_toe
pip install streamlit
streamlit run app.py
```

### Task 3 — Image Captioning (BLIP Transformers)
```bash
cd task3_image_captioning
pip install -r requirements.txt
streamlit run app.py
# Note: Downloads BLIP model (~450MB) on first run
```

### Task 4 — Recommendation System (TF-IDF + SVD)


### Task 5 — Face Detection & Recognition (OpenCV + dlib)
